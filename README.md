# student-performance-analysis
Student Performance Analytics System using Python, Pandas, NumPy, and Matplotlib to analyze marks, attendance, grades, and visualize student performance insights.

# Student Performance Analytics System

## Objective
The main objective of this project is to analyze student performance using marks and attendance data.

This project helps understand:
- Student performance in different subjects
- Effect of attendance on marks
- Top-performing and low-performing students

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

---

## Dataset Description
The dataset contains information of 30 students with the following columns:

- Student Name
- Math Marks
- Science Marks
- English Marks
- Attendance

---

## Analysis Performed

### Data Analysis
- Displayed first and last 5 records
- Checked dataset shape and column names
- Checked missing values and duplicate records
- Verified data types

### Basic Calculations
- Average marks of each subject
- Maximum and minimum marks
- Standard deviation

### Additional Analysis
- Created Total Marks column
- Identified top-performing student
- Identified lowest-performing student
- Found subject with highest average marks
- Assigned grades (A, B, C, D)
- Ranked students based on performance

---

## Visualizations
- Bar Chart → Average marks of each subject
- Histogram → Distribution of marks
- Pie Chart → Attendance distribution
- Scatter Plot → Attendance vs Total Marks
